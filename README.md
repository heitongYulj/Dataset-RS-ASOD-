# Dataset-RS-ASOD-
RS-ASOD is based on WorldView, GeoEye, and Landsat remote sensing images, which include various types of backgrounds such as forests, oceans, clouds, cities, and also contain aircraft instances in both parked and flying states. The dataset consists of 7,400 remote sensing images, with 24,477 aircraft instances. The image size is 640Ã—640 pixels, and the resolution ranges from 2 to 5 meters.
